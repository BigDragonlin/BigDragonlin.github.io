# 4.2HTTP1.0和HTTP1.1的区别

HTTP1.0是非持久性连接；一次请求对应一次TCP连接；不支持虚拟主机名

HTTP1.1是持久性连接；一个页面对应一次TCP连接；支持虚拟主机名
