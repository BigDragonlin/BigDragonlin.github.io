# Inspect窗口可编辑的变量

  1. 基本公共变量

任何定义为public的变量都会自动出现在Inspector窗口中。例如：

public class PlayerController : MonoBehaviour
{
    public float speed = 10.0f;
    public int health = 100;
}

在这个例子中，speed和health变量将在Inspector窗口中显示，开发者可以在Unity编辑器中直接修改它们的值。

  1. 使用SerializeField属性

如果你想保持变量的私有性，但仍希望它在Inspector窗口中可见，可以使用SerializeField属性。这允许你在保持封装的同时编辑变量：

public class PlayerController : MonoBehaviour
{
    [SerializeField]
    private float speed = 10.0f;
}
1
2
3
4
5

这里的speed变量虽然是私有的，但由于SerializeField属性，它仍然会出现在Inspector窗口中。

  1. 使用Range属性

你可以通过使用Range属性来限制数值变量在Inspector中的可能值，这对于创建易于理解的接口非常有帮助：

public class PlayerController : MonoBehaviour
{
    [Range(1, 100)]
    public int health = 50;
}
1
2
3
4
5

这样设置后，在Inspector窗口中，health变量将显示为一个滑块，其值被限制在1到100之间。

  1. 使用Header和Tooltip属性

为了使Inspector窗口更加清晰易懂，你可以使用Header和Tooltip属性来为变量添加标题和鼠标悬停文本：

public class PlayerController : MonoBehaviour
{
    [Header("Player Settings")]
    [Tooltip("Speed of the player.")]
    public float speed = 10.0f;

    [Tooltip("Health of the player.")]
    public int health = 100;
}

  1. 自定义数据类型

你也可以在Inspector中显示自定义的数据类型。例如，定义一个包含多个字段的类，并将其作为变量类型：

[System.Serializable]
public class Weapon
{
    public string name;
    public int damage;
}

public class PlayerController : MonoBehaviour
{
    public Weapon weapon;
}
