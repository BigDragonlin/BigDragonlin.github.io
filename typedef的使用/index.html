            typedef 在 C++ 中是一个关键字，用于为现有的类型创建一个新的名字（别名）。 这个机制属于 C++ 的类型系统的一部分，允许程序员给类型起一个新的、可能更简短或更具描述性的名字。 使用 typedef 可以提高代码的可读性和可维护性，特别是在处理复杂的数据类型（如指针、数组、结构体等）时。 下面是关于如何使用 typedef 的具体说明：

            # 1. 基本用法

            typedef 最基本的用法是为类型定义一个新名字。 例如，你可以为 unsigned long long 定义一个简短的别名 ull：

            typedef unsigned long long ull;
ull a = 1234567890123456789ULL;

            # 2. 结构体和类

            在 C++ 中，typedef 经常用来为结构体和类定义别名，使得在声明对象时不需要重复使用 struct 或 class 关键字。

            typedef struct {
int x, y;
} Point;

Point p1, p2;

            # 3. 指针类型

            对于指针类型，typedef 也可以简化声明，尤其是函数指针或数组指针。

            typedef int* IntPtr;
IntPtr ptr; // 等同于 int* ptr;

            # 4. 函数指针

            typedef 可以使函数指针的声明更加简洁明了。

            typedef void (*FuncPtr)(int, double);
FuncPtr myFunc;

            # 5. 模板别名（C++11 及之后）

            从 C++11 开始，可以使用 using 关键字来定义类型的别名，这在模板编程中尤其有用。

            template<typename T>
using Ptr = T*;

Ptr<int> intPtr; // 等同于 int* intPtr;
