# 单例模式（Singleton）

# 单例模式概述：

单例模式确保一个类只有一个实例，并提供一个全局访问点来访问这个实例。这在需要确保全局只有一个对象时非常有用。

# 实现步骤：

# 类的定义：

# 定义一个名为Singleton的类。

  • 私有的静态指针instance，初始值为nullptr，用于指向类的实例。
  • 私有的构造函数，防止外部直接创建实例。
  • 公有的静态方法getInstance()，用于获取类的实例。

# 方法实现：

  • getInstance()方法：
  • 检查instance是否为nullptr。
  • 如果是，创建一个新的Singleton对象。
  • 返回instance。
  • 线程安全（根据需要选择）：

为了确保线程安全，可以在getInstance()方法中添加锁。 这一步在多线程应用中非常关键，但会增加一些开销。 防止复制和赋值：

删除或私有化复制构造函数和赋值运算符，防止实例被复制。 考虑如何安全地销毁单例（如果需要）。

# 基础版本单例模式

#include <iostream>

class Singleton {
private:
    static Singleton* instance; // 静态私有实例
    Singleton() {} // 私有构造函数

public:
    // 禁止复制和赋值
    Singleton(const Singleton&) = delete;
    Singleton& operator=(const Singleton&) = delete;

    // 获取单例对象的静态方法
    static Singleton* getInstance() {
        if (instance == nullptr) {
            instance = new Singleton();
        }
        return instance;
    }
};

// 初始化静态成员变量
Singleton* Singleton::instance = nullptr;

int main() {
    // 获取单例对象
    Singleton* singletonInstance = Singleton::getInstance();

    std::cout << "Singleton instance address: " << singletonInstance << std::endl;
    // 尝试再次获取单例对象，验证是否为相同实例
    Singleton* anotherInstance = Singleton::getInstance();
    std::cout << "Another Singleton instance address: " << anotherInstance << std::endl;

    return 0;
}

# 懒汉式（线程安全版本）

在这个实现中，单例实例在第一次被需要时创建。为了保证线程安全，通常会在获取实例的方法中加锁。

class Singleton {
private:
    static Singleton* instance;
    Singleton() {}
public:
    static Singleton* getInstance() {
        if (instance == nullptr) { // 双重检查锁定
            std::lock_guard<std::mutex> lock(mutex); // C++11及以上版本的线程安全锁
            if (instance == nullptr) {
                instance = new Singleton();
            }
        }
        return instance;
    }

private:
    static std::mutex mutex; // 用于同步的互斥锁
};

# 饿汉式（线程安全版本）

在这个实现中，单例实例在程序启动时就被创建好了。由于实例是在程序启动时在主线程中创建的，因此自然是线程安全的。

class Singleton {
private:
    Singleton() {}
public:
    static Singleton& getInstance() {
        static Singleton instance; // 局部静态变量
        return instance;
    }
};

# 使用std::call_once和std::once_flag（C++11及以上）

class Singleton {
private:
    static Singleton* instance;
    Singleton() {}
    static std::once_flag flag;

public:
    static Singleton* getInstance() {
        std::call_once(flag, [](){ instance = new Singleton(); });
        return instance;
    }
};

Singleton* Singleton::instance = nullptr;
std::once_flag Singleton::flag;
