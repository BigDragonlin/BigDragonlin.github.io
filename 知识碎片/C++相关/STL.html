# 基础输入输出（I/O）

# 理论

C++标准库提供了输入输出操作的流类。常用的流类有：

  • std::cin：标准输入流，用于从键盘读取输入。
  • std::cout：标准输出流，用于向屏幕输出内容。
  • std::cerr：标准错误流，用于输出错误信息。

这些流类都定义在<iostream>头文件中。

# 代码示例

让我们通过几个简单的例子来熟悉这些流类的使用。

# 示例1：读取并输出整数
#include <iostream>

int main() {
    int number;
    std::cout << "请输入一个数字: ";
    std::cin >> number; // 从标准输入读取一个整数
    std::cout << "您输入的数字是: " << number << std::endl; // 输出读取的整数
    return 0;
}

# 容器（Containers）

容器是用来存储数据的通用数据结构。STL提供了多种类型的容器，如 vector、list、deque、set、map等。

  • Vector：向量，可动态增长的数组。
  • List：列表，双向链表。
  • Deque：双端队列。
  • Set：集合，存储唯一元素，自动排序。
  • Map：映射，存储键值对，键唯一，自动排序。

# 双端队列（deque）

deque是一个双端队列，支持在两端快速插入和删除。

# 特性：

  • 动态数组
  • 支持随机访问
  • 在两端插入和删除元素都很高效

# 示例代码：

#include <iostream>
#include <deque>

void dequeExample() {
    std::deque<int> deq = {1, 2, 3, 4, 5};
    deq.push_back(6); // 在末尾添加元素
    deq.push_front(0); // 在开头添加元素

    for(int i = 0; i < deq.size(); ++i) {
        std::cout << deq[i] << " "; // 随机访问
    }
    std::cout << std::endl;

    deq.pop_back(); // 移除末尾元素
    deq.pop_front(); // 移除开头元素

    for(auto it = deq.begin(); it != deq.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;
}

int main() {
    dequeExample();
    return 0;
}

# 集合（set）和多重集合（multiset）

set是一种有序集合，不允许重复元素；multiset允许重复元素。

# 特性：

  • 有序集合
  • 快速查找、插入和删除

#include <iostream>
#include <set>

void setExample() {
    std::set<int> s = {5, 3, 1, 4, 2};

    for(const auto& elem : s) {
        std::cout << elem << " "; // 有序输出
    }
    std::cout << std::endl;

    s.erase(4); // 删除元素

    for(const auto& elem : s) {
        std::cout << elem << " ";
    }
    std::cout << std::endl;
}

int main() {
    setExample();
    return 0;
}

# 映射（map）和多重映射（multimap）

map是一种关联容器，存储键值对；multimap允许重复键。

# 特性：

  • 键值对存储

#include <iostream>
#include <map>

void mapExample() {

    m[3] = "three";

    for(const auto& pair : m) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }

    m.erase(2); // 删除键值对

    for(const auto& pair : m) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
}

int main() {
    mapExample();
    return 0;
}

# 总结

std::multimap 是一个非常有用的容器，当你需要一个键对应多个值时，它能很好的解决这个问题。通过插入、查找、遍历和删除元素，可以有效地管理数据。

以下是一个完整的示例代码：

#include <iostream>
#include <map>

class BuildTask {
    // 假设BuildTask类已经定义
};

int main() {
    std::multimap<unsigned int, BuildTask*, std::less<unsigned int>> BuildTaskMap;

    BuildTask* task1 = new BuildTask();
    BuildTask* task2 = new BuildTask();
    BuildTask* task3 = new BuildTask();

    BuildTaskMap.insert(std::make_pair(1, task1));
    BuildTaskMap.insert(std::make_pair(1, task2));
    BuildTaskMap.insert(std::make_pair(2, task3));

    for (const auto& pair : BuildTaskMap) {
        std::cout << "Key: " << pair.first << ", Value: " << pair.second << std::endl;
    }

    auto range = BuildTaskMap.equal_range(1);
    for (auto it = range.first; it != range.second; ++it) {
        std::cout << "Key: " << it->first << ", Value: " << it->second << std::endl;
    }

    BuildTaskMap.erase(1);

    return 0;
}

# std::bitset

好的，让我们一步步来学习 std::bitset 的使用。std::bitset 是一个用于处理固定大小的位序列的容器类。它提供了一组方便的成员函数来操作位。我们将通过几个示例逐步介绍 std::bitset 的功能和使用方法。

# 1. 基本用法

std::bitset 的大小在编译时确定，因此需要在模板参数中指定大小。例如，std::bitset<8> 表示一个包含 8 位的位集合。

#include <iostream>
#include <bitset>

int main() {
    std::bitset<8> bits; // 默认初始化为 00000000
    std::cout << "Initial bits: " << bits << std::endl;

    std::bitset<8> bitsFromInt(42); // 使用整数初始化
    std::cout << "Bits from integer 42: " << bitsFromInt << std::endl;

    std::bitset<8> bitsFromString("1100"); // 使用字符串初始化
    std::cout << "Bits from string '1100': " << bitsFromString << std::endl;

    return 0;
}

# 2. 位操作

std::bitset 提供了一些常用的位操作函数，如 set、reset、flip 和 test。

#include <iostream>
#include <bitset>

int main() {
    std::bitset<8> bits;

    bits.set(1); // 设置第 1 位为 1
    std::cout << "After setting bit 1: " << bits << std::endl;

    bits.set(); // 将所有位设置为 1
    std::cout << "After setting all bits: " << bits << std::endl;

    bits.reset(1); // 将第 1 位重置为 0
    std::cout << "After resetting bit 1: " << bits << std::endl;

    bits.reset(); // 将所有位重置为 0
    std::cout << "After resetting all bits: " << bits << std::endl;

    bits.flip(2); // 翻转第 2 位
    std::cout << "After flipping bit 2: " << bits << std::endl;

    bits.flip(); // 翻转所有位
    std::cout << "After flipping all bits: " << bits << std::endl;

    std::cout << "Bit 2 is " << (bits.test(2) ? "set" : "not set") << std::endl;

    return 0;
}

    std::bitset<8> newBits(bitString); // 从字符串初始化
    std::cout << "New bits from string: " << newBits << std::endl;

    return 0;
}

# 迭代器（Iterators）

迭代器是一种访问容器中元素的对象，类似于指针。根据容器的类型和需要，迭代器分为多种类型，如输入迭代器、输出迭代器、前向迭代器、双向迭代器、随机访问迭代器等。

# 迭代器的类型

  1. 输入迭代器（Input Iterator）

    • 只能读取一次值，不能进行写操作。
    • 支持++（前缀和后缀）和解引用操作*。
    • 主要用于一次性遍历，如从输入流读取数据。

  2. 输出迭代器（Output Iterator）

    • 只能写入一次值，不能进行读操作。
    • 支持++（前缀和后缀）和解引用操作*。
    • 主要用于一次性写入，如向输出流写数据。

  3. 前向迭代器（Forward Iterator）

    • 可以进行多次读写操作。
    • 支持++（前缀和后缀）和解引用操作*。
    • 适用于单向遍历且允许多次遍历的场景。

  4. 双向迭代器（Bidirectional Iterator）

    • 前向迭代器的扩展，允许在两个方向上遍历。
    • 支持++（前缀和后缀）、--（前缀和后缀）和解引用操作*。
    • 适用于需要前后遍历的容器，如list。

  5. 随机访问迭代器（Random Access Iterator）

    • 最强大的迭代器，支持所有双向迭代器的操作外，还支持跳跃式访问。
    • 支持++、--、[]、+、-等操作。
    • 适用于数组、向量等可随机访问的容器，如vector和deque。

# 常用的STL迭代器

以下是几个常用的STL迭代器的例子：

#include <iostream>
#include <vector>
#include <list>
#include <iterator>

int main() {
    // 使用 vector 作为示例
    std::vector<int> vec = {1, 2, 3, 4, 5};

    // 使用随机访问迭代器遍历 vector
    for (std::vector<int>::iterator it = vec.begin(); it != vec.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    // 使用 list 作为示例
    std::list<int> lst = {1, 2, 3, 4, 5};

    // 使用双向迭代器遍历 list
    for (std::list<int>::iterator it = lst.begin(); it != lst.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    return 0;
}

# 算法（Algorithms）

算法是作用于容器的函数，用于执行诸如搜索、排序、计数、遍历等操作。STL的算法设计为与容器无关，可以通过迭代器与任何容器配合使用。 C++标准模板库（STL）提供了丰富的算法库，这些算法与STL容器和迭代器紧密结合，可以在容器上执行各种操作。以下是一些常见且有用的STL算法：

# std::find

# C标准库常用函数

以下是一些常用的 C 标准库函数，它们被广泛用于各种编程任务，包括字符串处理、内存管理、输入输出操作等。

# 字符串处理函数（ 或 <string.h>）

  1. strlen(const char *str)

    • 返回字符串 str 的长度，不包括终止符 \0。

  2. strcpy(char *dest, const char *src)

    • 将字符串 src 复制到 dest。

  3. strncpy(char *dest, const char *src, size_t n)

    • 将最多 n 个字符从字符串 src 复制到 dest。

  4. strcat(char *dest, const char *src)

    • 将字符串 src 连接到 dest 末尾。

  5. strncat(char *dest, const char *src, size_t n)

    • 将最多 n 个字符从字符串 src 连接到 dest 末尾。

  6. strcmp(const char *str1, const char *str2)

    • 比较字符串 str1 和 str2，返回值小于、等于或大于零，分别表示 str1 小于、等于或大于 str2。

  7. strncmp(const char *str1, const char *str2, size_t n)

    • 比较最多 n 个字符的字符串 str1 和 str2。

# 内存管理函数（ 或 <stdlib.h>）

  1. malloc(size_t size)

    • 分配 size 字节的内存，返回指向已分配内存的指针。

  2. calloc(size_t num, size_t size)

    • 分配内存并初始化为零。分配 num 个元素，每个元素 size 字节。

  3. realloc(void *ptr, size_t size)

    • 重新分配内存块，调整为 size 字节。

  4. free(void *ptr)

    • 释放先前分配的内存。
