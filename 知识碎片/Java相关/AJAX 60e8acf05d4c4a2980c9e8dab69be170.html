# 什么是ajax

ajax不是一门新的编程语言，而是一系列编程语言的混用，比如html，javascript，dom，xml。主要是xml和JavaScript

ajax：asynchronous JavaScript and XML

asynchronous ： 异步的意思

JavaScript：使用脚本对象XMLRequest发送请求，接受响应数据

xml：发送和接受响应数据的一种格式，现在使用的是jason
