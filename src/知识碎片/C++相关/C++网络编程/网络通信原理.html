网络通信原理涉及的是如何在不同计算机间传递数据。核心概念包括TCP/IP协议栈，它是现代网络通信的基石。

# TCP/IP协议栈

  1. 应用层 (Application Layer): 这一层为应用软件提供网络服务。包含的协议如HTTP（超文本传输协议），FTP（文件传输协议），SMTP（简单邮件传输协议）等。

  2. 传输层 (Transport Layer): 提供端到端的数据传输服务。核心协议有TCP（传输控制协议）和UDP（用户数据报协议）。

  3. 网络层 (Network Layer): 负责数据包从源头到目的地的传递和路由选择。主要协议是IP（互联网协议）。

  4. 链路层 (Link Layer): 负责在网络设备之间传输数据。它包括数据的封装成帧和物理传输。

# TCP vs. UDP

  • TCP (Transmission Control Protocol): 面向连接的协议，提供可靠、有序和错误检测机制的数据传输服务。
  • UDP (User Datagram Protocol): 无连接协议，提供一种不可靠的服务来发送数据报文。

# 套接字编程 (Socket Programming)

套接字编程是实现网络通信的基础方法。在C++中，套接字是一个端点，用于发送和接收数据。

  • 创建套接字 (Socket Creation): 创建一个套接字以开始网络通信。
  • 绑定套接字 (Bind Socket): 将套接字绑定到一个本地地址和端口。
  • 监听连接 (Listen for Connections): 仅限TCP，服务器监听来自客户端的连接请求。
  • 接受连接 (Accept Connections): 仅限TCP，服务器接受客户端的连接请求。
  • 发送和接收数据 (Send and Receive Data): 通过套接字发送和接收数据。
  • 关闭套接字 (Close Socket): 结束通信时关闭套接字。
