# 问题

为了解决love编写过程中没有代码提示的问题。

# 解决方法

使用插件EmmyLua + Emmy-love-api解决

  • 下载getEmmyApi.lua文件 (opens new window)

  • 下载或者clone love-api (opens new window)
  • 将getEmmyApi.lua文件放到下载的文件夹，创建api文件夹

  • 配置好lua运行环境，执行getEmmyApi.lua

  • 生成成功

  • 将生成的注解移动到项目中

  • 就可以有代码提示出现了
