# 使用 LuaRocks 管理和加载 Lua 模块

LuaRocks 是 Lua 的包管理器，用于安装和管理 Lua 模块。在这篇文章中，我们将介绍如何使用 LuaRocks 安装和管理 Lua 模块，如何设置环境变量以确保 Lua 能够正确加载这些模块，并展示一些常用模块（如 lua-cjson）的使用示例。此外，我们还将解释一些 LuaRocks 相关目录的含义。

# 1. 安装 LuaRocks

你可以从 LuaRocks 官方网站 (opens new window) 下载并安装 LuaRocks。安装完成后，可以通过以下命令检查是否安装成功：

luarocks --version
1

# 2. 安装 Lua 模块

使用 LuaRocks 安装 Lua 模块非常简单。以下是安装 lua-cjson 模块的示例：

luarocks install lua-cjson
1

# 3. 设置环境变量

为了确保 Lua 加载器能够找到通过 LuaRocks 安装的模块，我们需要设置 LUA_PATH 和 LUA_CPATH 环境变量。这些变量定义了 Lua 脚本和 C 模块的查找路径。

# 在 Windows 上设置环境变量

  1. 打开系统属性，进入“高级”选项卡。

  2. 点击“环境变量”。

  3. 在“系统变量”中，找到并编辑 LUA_PATH 和 LUA_CPATH 变量。如果没有这些变量，点击“新建”来添加。

  4. 设置 LUA_PATH 变量值为：

    C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\share\\lua\\5.1\\?.lua;C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\share\\lua\\5.1\\test\\?.lua;;
    1

  5. 设置 LUA_CPATH 变量值为：

    C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\lib\\lua\\5.1\\?.dll;;
    1

# 4. 在 Lua 脚本中动态设置路径

你也可以在 Lua 脚本中手动设置 package.path 和 package.cpath：

-- 设置 Lua 模块的路径
package.path = package.path .. ";C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\share\\lua\\5.1\\?.lua;C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\share\\lua\\5.1\\test\\?.lua;;"
package.cpath = package.cpath .. ";C:\\Program Files (x86)\\Lua\\luarocks-3.9.1\\systree\\lib\\lua\\5.1\\?.dll"

-- 引用 lua-cjson 模块
local cjson = require "cjson"

-- 使用 cjson 模块
local lua_table = { key = "value" }
local json_str = cjson.encode(lua_table)
print("JSON String: ", json_str)

local decoded_table = cjson.decode(json_str)
print("Decoded Lua Table: ", decoded_table.key)

# 5. 加载某个文件夹及其子文件夹下的 .lua 文件

为了加载某个文件夹及其子文件夹下的 .lua 文件，可以编写一个递归函数遍历该文件夹，并将所有 .lua 文件路径添加到 package.path 中：

local lfs = require "lfs"  -- LuaFileSystem 库，用于遍历文件夹

-- 递归遍历文件夹并收集 .lua 文件路径
local function collect_lua_files(folder)
    local lua_files = {}
    
    local function scan_dir(dir)
        for file in lfs.dir(dir) do
            if file ~= "." and file ~= ".." then
                local full_path = dir .. "/" .. file
                local attr = lfs.attributes(full_path)
                if attr.mode == "directory" then
                    scan_dir(full_path)  -- 递归调用，遍历子文件夹
                elseif file:match("%.lua$") then
                    table.insert(lua_files, full_path)
                end
            end
        end
    end

    scan_dir(folder)
    return lua_files
end

-- 转换文件路径为 Lua 模块路径
local function convert_to_module_paths(lua_files)
    local module_paths = {}
    for _, file in ipairs(lua_files) do
        local module_path = file:gsub("/", "."):gsub("\\", "."):gsub("%.lua$", "")
        table.insert(module_paths, module_path)
    end
    return module_paths
end

-- 将文件夹及其子文件夹中的 .lua 文件添加到 package.path 中
local function add_to_package_path(folder)
    local lua_files = collect_lua_files(folder)
    local module_paths = convert_to_module_paths(lua_files)
    for _, module_path in ipairs(module_paths) do
        package.path = package.path .. ";" .. module_path
    end
end

-- 示例：将指定文件夹及其子文件夹中的 .lua 文件添加到 package.path 中
local folder_to_add = "C:/path/to/your/lua/folder"
add_to_package_path(folder_to_add)

# 6. LuaRocks 相关目录含义

当你使用 LuaRocks 安装模块时，模块文件会被放置在特定的目录中。以下是一些重要目录及其含义：

  1. C:\Program Files (x86)\Lua\luarocks-3.9.1\systree\lib\luarocks\rocks-5.1

    这个目录包含了通过 LuaRocks 安装的各个 Lua 模块及其相关文件。每个模块都有自己独立的文件夹，文件夹内包含模块的源代码、构建脚本和文档。

    C:\Program Files (x86)\Lua\luarocks-3.9.1\systree\lib\luarocks\rocks-5.1\
├── 2048.nvim
├── lua-cjson
├── luafilesystem
├── luaprofiler
└── manifest
    1
    2
    3

    6
    • 2048.nvim: 可能是一个 Neovim 插件或 Lua 模块，用于在 Neovim 编辑器中运行2048游戏或相关功能。
    • lua-cjson: 包含 lua-cjson 模块的源代码、文档和构建文件。
    • luafilesystem: 包含 LuaFileSystem 模块的源代码、文档和构建文件。
    • luaprofiler: 包含 LuaProfiler 模块的源代码、文档和构建文件。
    • manifest: 记录了通过 LuaRocks 安装的所有模块的信息，是一个清单文件，用于跟踪安装的模块及其版本。

  2. C:\Program Files (x86)\Lua\luarocks-3.9.1\systree\share\lua\5.1

    这个目录包含了通过 LuaRocks 安装的 Lua 脚本和模块。以下是这个目录的截图和解释：

├── 2048
├── cjson
├── json2lua.lua
└── lua2json.lua

    • 2048: 可能包含与 2048 游戏相关的 Lua 脚本和资源文件。
    • cjson: 包含 lua-cjson 模块的 Lua 脚本文件，提供核心功能用于 JSON 编码和解码。
    • json2lua.lua: 用于将 JSON 数据转换为 Lua 表的 Lua 脚本。
    • lua2json.lua: 用于将 Lua 表转换为 JSON 数据的 Lua 脚本。

  3. C:\Program Files (x86)\Lua\luarocks-3.9.1\systree\lib\lua\5.1

    这个目录包含了通过 LuaRocks 安装的 C 模块的二进制文件。以下是这个目录的截图和解释：

├── cjson.dll
├── lfs.dll
└── profiler.dll

    • lfs.dll: LuaFileSystem 模块的二进制文件，用于文件系统操作。
