# Obsidian 入门指南

欢迎来到 Obsidian 的世界！Obsidian 不仅仅是一个笔记软件，它更是一个第二大脑构建工具。它基于本地 Markdown 文件，强调“链接”而非“分类”，帮助你构建知识网络。

# 1. 核心理念

  • 本地优先：你的所有数据都保存在本地文件夹中（称为 Vault），你拥有数据的完全掌控权。
  • Markdown：使用通用的纯文本格式，不被特定软件锁定。
  • 双向链接：通过链接将笔记关联起来，模拟大脑的联想记忆。

# 2. 基础操作

# 2.1 创建与链接

  • 新建笔记：Ctrl/Cmd + N。
  • 双向链接：这是 Obsidian 的灵魂。输入 [[ 即可唤起链接面板，选择已有笔记或输入新名称创建链接。例如：[[我的第一篇笔记]]。
  • 内部引用：输入 ![[ 可以直接在当前文档中显示被链接笔记的内容（嵌入）。

# 2.2 常用 Markdown 语法

Obsidian 使用标准的 Markdown 语法：

  • 标题：使用 # 号。
    • # 一级标题
    • ## 二级标题
  • 列表：使用 - (减号加空格)。
  • 粗体：使用 **文字**。
  • 斜体：使用 *文字*。
  • 引用：使用 > 文字。
  • 任务列表：使用 - [ ] 待办事项。

# 2.3 标签 (Tags)

  • 使用 #标签名 来给笔记分类。例如 #学习 或 #待办。
  • 标签适合跨文件夹的属性标记。

# 3. 界面功能

  • 文件列表：左侧显示你的文件夹和文件结构。
  • 关系图谱 (Graph View)：点击左侧的“关系图谱”图标，可以看到笔记之间是如何连接的。这是 Obsidian 最酷的功能之一。
  • 命令面板：按下 Ctrl/Cmd + P，可以搜索并执行 Obsidian 的所有命令（无需记忆快捷键）。

# 4. 进阶功能（插件系统）

Obsidian 的强大在于其可扩展性。

  • 核心插件：官方自带，可以在设置中开启/关闭。

    • 日记 (Daily Notes)：每天自动创建一个以日期命名的笔记，适合记录流水账或每日反思。
    • 白板 (Canvas)：无限大的画布，可以放入卡片、图片、网页，适合头脑风暴。
    • 模板 (Templates)：快速插入预定义的文本格式。

  • 社区插件：由用户开发（需要在设置中关闭“安全模式”以使用）。

    • Dataview：将你的 Vault 变成数据库，动态查询笔记。
    • Excalidraw：在 Obsidian 里画图。
    • Calendar：在侧边栏显示日历视图。

# 5. 新手建议

  1. 不要过度整理：刚开始不要纠结于复杂的文件夹结构。
  2. 多用链接：遇到相关的概念，就用 [[ ]] 链接起来。
  3. 从日记开始：开启“日记”插件，每天记录一点想法，慢慢积累。
  4. 渐进式学习：不要试图一次性学会所有插件，按需学习。
