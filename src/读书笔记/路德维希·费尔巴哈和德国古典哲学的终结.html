# 20240317

相信一种哲学，不要掉入到怀疑论当中。

在读这本书的时候，总有一个主题萦绕在我的脑海，这本书到底对我的生活有什么用？在我浅薄的哲学知识中， 这本书中介绍的唯物辩证法和历史唯物主义相对于其它世界观更符合我对世界的“感觉”。奥卡姆剃刀原理说， 对某一现象进行解释的时候尽可能保证解释的简单性。相较于宗教的神秘主义，唯物辩证法的解释更加简单， 更加符合我的直觉。

人不是生活在真空当中，有人的地方就有环境，就有社会，我们就要去解释问题，去行动，去解决问题。 在这个过程中，我们需要一种世界观，一种方法论。尽管马克思的唯物辩证法和历史唯物主义并不是完美的， 但是它是我目前所知道最好的解释工具。马克思已经离开我们一百多年来，我们也不应该过分苛求他的理论能够 完美的解释现在的现实。

在心理学上，有一个概念叫做“自我实现预言”，这个概念是指一个人对自己的预期会影响到自己的行为，从而使得预期成为现实。 当我们相信矛盾是普遍的，运动是普遍的，当我们的事业遭受挫折的时候，我们会想到这是一种矛盾，是一种运动，这是一个 过程，我们会想到这个过程会有一个结果。这种思想会使我们更加坚定的去追求我们的目标，去解决问题。辩证法的的矛盾与运动， 量变质变，否定之否定，相互联系，相互转化不仅仅是一种哲学观点，更应该是一种相信，一种信念。

要有勇气投入到社会实践当中去。

在马克思主义中，实践是被认为理论认识和社会变革的核心，马克思主义强调理论必须和实践相结合。马克思主义对个人 的现实意义之一就是要求个人投入到社会实践中去，要求个人在社会实践中去改造世界。马克思主义认为，人的认识是通过 实践而产生的，人的认识是通过实践而发展的。

人总是难以跨越知道和做到之间的鸿沟，我们总是知道一些道理，却很难做到。在编程中，能“看懂”，和能“写出来”是两回事， 以我的经验，纯粹的读源码是很容易忘记，但是自己写出来的东西却很难忘记。最近在编程学习中，借助chatgpt，使得我的 效率格外高，因为我可以在编程的过程中，借助chatgpt，不断的和chatbot交流，问chatgpt我不清楚的细节， 这样我可以在编程的过程中，直接的应用到实践中去，在实践中学习才是一种快速的学习方式。

我知道在写完这些的时候，我还是很难将身边的事物都用辩证法去处理。但是我相信，我会慢慢的改变。 我希望自己有勇气去直面问题，尤其是重要的问题。 我很喜欢一部小说中的一段话：我们立于大雪弥漫、浓雾障眼的山口,我们只能偶尔瞥见未必正确的路径。我们待在那儿不动,就会被冻死; 若是误入歧途,就会摔得粉身碎骨。我们无法确知是否有一条正确的道路。我们该怎么做呢?“你们当刚强壮胆”,往最好处努力, 不要说谎,我们要睁大双眼,昂起头颅,走好脚下的路,不管它通向何方。如果死神终结了一切,我们也拿它没办法。如果事情不是这样, 那就以大丈夫气概坦然走进下一幕,无论它是什么样子,不要做巧舌之辩,也不要掩饰自己的真面目。

# 20240315

辩证哲学所承认的唯一普遍性的东西是：运动、变化、矛盾的普遍性。 书中提到“在它面前，除了生存和灭亡的不断过程、无止境的由低级到高级的不断过程，什么也不存在”。 从辩证法上来看，低级走向高级是从较低的生产力，社会组织形式，自由解放走向较高的生产力，社会组织形式，自由解放的过程。

这样一个过程真的是对全人类都是行之有效的么？社会发展到了今天，却还是有很多国家、很多人在遭受饥饿。 社会从低级走向高级，需要发展生产力，改革生产关系。这里的“社会”，在当今世界来看，更符合实际的是指的全球社会， 而不是某一个国家的社会。阿美莉卡生产力的发展，生产关系的变革，虽然整体上是对全人类有利的，但是在这个过程中， 却造成了很多国家、很多人的痛苦。

我毫不怀疑整个社会是向前发展、进步的，可是我们普通人呢？我们没有那么高尚，我们想获得的是自己的幸福。 整个社会再向前发展，如果公平分配社会发展的收益的话，整个社会所有人都会受益，但是现实中，社会发展的收益并不是公平分配的， 有部分人拿走了大部分的收益，而另一部分人却没有得到应有的收益。 如果这个时候再用辩证法的前途光明，道路曲折去解释的话，是不是有些过于残忍。

  • 认识世界是可能的。
  • 推动这些哲学家思想的发展的，不是纯粹的思想力量，还依靠自然科学和工业发展的进步
  • 形而上学的英文是metaphysics,meta是超越的意思，physics是自然的意思。中国有句古话“形而上谓之道，形而下谓之器”。 哲学的metaphysics是指的是“通过推理，概念分析，而非科学实验来解答问题”
  • 我们不仅生活在自然界中，而且生活在人类社会中。
  • 费尔巴哈没有将唯物主义哲学观推广到整个社会
  • 黑格尔说：恶推动了社会的进步
  • 皇宫中人们所想的和住在茅草屋人们所想的是不一样的，如果你因为饥饿，贫穷而身体中没有养料，那么你的头脑中便没有 道德所用的养料了。
  • 资产阶级的平等，即使是有，也只是比封建社会的平等好一点。
  • 现代历史已经证明，一切政治斗争都是阶级斗争。
  • 国家反映的上支配着生产的阶级的经济需要。
  • 被统治阶级反对统治阶级的斗争，首先反映的是被统治阶级返回统治阶级的政治斗争。
  • 任何意识形态一经产生，就通现有的观念材料相结合发展起来，并对这些观念材料加以改造。
