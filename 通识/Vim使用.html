      Vim使用

      上下左右移动

      • h: 左
      • j: 下
      • k: 上
      • l: 右

      一个单词一个单词移动

      • w: 向前移动一个单词
      • b: 向后移动一个单词
      • e: 移动到单词末尾
      • ge: 移动到前单词末尾
      • f + char: 向后查找字符
      • F' + char: 向前查找字符
      • ;: 重复上一次f命令
      • ,: 重复上一次F命令

      跳转到定义

      • gd: 跳转到定义

      复制和粘贴，剪切和粘贴

      • y: 复制
      • p: 粘贴
      • x: 剪切
      • yy: 复制一行
      • dd: 剪切一行

      复制粘贴的时候要进入visual模式，v进入字符选择模式，V进入行选择模式，Ctrl + v进入块选择模式

      整段选择

      • {: 选择上一段
      • }: 选择下一段
      • ctrl + d: 选择下一篇
      • ctrl + u: 选择上一篇

      撤销和重做

      • u: 撤销
      • Ctrl + r: 重做
      • .: 重复上一次操作

      查找和替换

      • /: 查找
      • n: 下一个
      • N: 上一个

      命令 + 数字 + 动作

      • 3w: 向前移动三个单词
      • 3j: 向下移动三行
      • d3w: 删除三个单词
      • c3w: 修改三个单词

      插入

      • i: 在当前字符前插入
      • a: 在当前字符后插入
      • I: 在当前行首插入
      • A: 在当前行尾插入
      • o: 在当前行下插入
      • O: 在当前行上插入

      删除

      • x: 删除当前字符
      • X: 删除前一个字符
      • d: 删除
      • dd: 删除一行
      • D: 删除到行尾
      • C: 删除到行尾并进入插入模式
      • s: 删除当前字符并进入插入模式
      • S: 删除当前行并进入插入模式
      • ctrl + w: 删除前一个单词
      • ctrl + u: 删除到行首
      • ctrl + k: 删除到行尾
      • ctrl + h: 删除前一个字符
      • ctrl + d: 删除下一个字符
      • ctrl + y: 删除上一行
      • ctrl + e: 删除下一行
      • d/hello: 删除到hello字符

      插入

      • i: 在当前字符前插入
      • a: 在当前字符后插入
      • I: 在当前行首插入
      • A: 在当前行尾插入
      • o: 在当前行下插入
      • O: 在当前行上插入
      • r: 替换当前字符
      • R: 替换当前字符
      • c: 修改
      • cc: 修改一行
      • C: 修改到行尾
      • s: 修改当前字符
      • S: 修改当前行

      返回上次编辑位置

      • gi: 返回上次编辑位置
      • g;: 返回上次编辑位置

      改变

      • cw: 改变一个单词
      • cb: 改变前一个单词
      • ce: 改变到单词末尾
      • cge: 改变到前单词末尾

      visual模式 视觉模式

      • v: 字符选择
      • V: 行选择
      • Ctrl + v: 块选择
      • gv: 重新选择上次选择的区域

      移动到行的开头和结尾

      • 0: 移动到行首
      • $: 移动到行尾

      移动到方法的开头和末尾

      • [[: 移动到方法开头
      • ]]: 移动到方法末尾

      选中这个单词

      • viw: 选中这个单词
        v是进入visual模式
        i是表示insert或者inner
        w是表示单词

      移动到首行

      • gg: 移动到首行
        第一个g是goto的意思
      • G: 移动到末行
