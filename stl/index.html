      STL主要由三大组件构成：容器（Containers）、迭代器（Iterators）、算法（Algorithms）。容器用于存储数据，迭代器用于访问和遍历容器中的数据，算法则可以在数据结构上执行各种操作。

      # 1. 容器（Containers）

      容器是用来存储数据的通用数据结构。STL提供了多种类型的容器，如 vector、list、deque、set、map等。

      • Vector：向量，可动态增长的数组。
      • List：列表，双向链表。
      • Deque：双端队列。
      • Set：集合，存储唯一元素，自动排序。
      • Map：映射，存储键值对，键唯一，自动排序。

      # vector的更多用法

      std::vector 是 C++ STL 中最常用的序列容器之一，它提供了一种灵活且易于使用的方式来存储和操作动态数组。这里，我将介绍 std::vector 的更多用法，包括如何初始化向量、访问元素、修改容器内容、以及其他常用操作。

      # 初始化 vector

      #include <vector>
std::vector<int> vec1; // 空的vector
std::vector<int> vec2(5, 10); // 大小为5，每个元素初始化为10
std::vector<int> vec3{1, 2, 3, 4, 5}; // 初始化列表
std::vector<int> vec4 = {1, 2, 3, 4, 5}; // 同上，使用赋值初始化
std::vector<int> vec5(vec3.begin(), vec3.end()); // 使用迭代器范围初始化
std::vector<int> vec6(vec3); // 复制vec3

      # 访问元素

      std::vector<int> vec = {1, 2, 3, 4, 5};

// 使用下标访问
std::cout << "第一个元素: " << vec[0] << std::endl;

// 使用at()访问，at()会检查下标范围
std::cout << "第二个元素: " << vec.at(1) << std::endl;

// 使用front()获取第一个元素
std::cout << "第一个元素: " << vec.front() << std::endl;

// 使用back()获取最后一个元素
std::cout << "最后一个元素: " << vec.back() << std::endl;

// 使用data()获取指向第一个元素的指针
int* ptr = vec.data();
std::cout << "第一个元素: " << *ptr << std::endl;

      # 修改容器内容

      std::vector<int> vec;

// 添加元素

// 改变vector大小
vec.resize(10); // 将大小改变为10

      # 容量和大小操作

      # list的更多用法

      std::list 是 C++ 标准模板库（STL）中提供的一个容器，它实现了一个双向链表。 与 std::vector 相比，list 提供了在任何位置插入和删除元素的高效操作， 但不支持直接通过下标访问元素。这里，我将介绍 std::list 的一些基本用法， 包括如何初始化列表、访问元素、修改列表内容，以及执行其他常见操作。

      # 初始化 list

      #include <list>

std::list<int> list1; // 创建一个空的list
std::list<int> list2(5, 10); // 创建一个包含5个元素的list，每个元素的值都是10
std::list<int> list3{1, 2, 3, 4, 5}; // 使用初始化列表创建list
std::list<int> list4 = {1, 2, 3, 4, 5}; // 同上，使用赋值初始化

      # 访问元素

      由于 std::list 不支持随机访问，所以不能像 vector 那样使用下标运算符 [] 或 at() 方法来访问元素。不过，你可以使用迭代器或 front() 和 back() 成员函数来访问第一个和最后一个元素。

      std::list<int> myList{1, 2, 3, 4, 5};

// 使用front()获取第一个元素
std::cout << "第一个元素: " << myList.front() << std::endl;

// 使用back()获取最后一个元素
std::cout << "最后一个元素: " << myList.back() << std::endl;

      # 修改列表内容

      std::list<int> myList;

// 添加元素
myList.push_back(6); // 在列表尾部添加一个元素
myList.push_front(0); // 在列表头部添加一个元素

// 在指定位置插入元素
auto it = myList.begin();
std::advance(it, 1); // 移动迭代器到指定位置
myList.insert(it, 2); // 在第二个位置插入元素2

// 删除元素
myList.pop_front(); // 删除第一个元素
myList.pop_back(); // 删除最后一个元素
myList.erase(myList.begin()); // 删除指定迭代器位置的元素

// 清空list
myList.clear();

      # 迭代器遍历

      由于 list 不支持随机访问，所以通常使用迭代器来遍历元素。

      std::list<int> myList = {1, 2, 3, 4, 5};

// 正向遍历list
for (auto it = myList.begin(); it != myList.end(); ++it) {
std::cout << *it << " ";
}
std::cout << std::endl;

// 反向遍历list
for (auto it = myList.rbegin(); it != myList.rend(); ++it) {
std::cout << *it << " ";
}
std::cout << std::endl;

      # 其他操作

      std::list 还提供了一系列用于排序、合并、翻转和去重的成员函数。

      cpp Copy code

      std::list<int> myList = {4, 1, 3, 2, 5};

// 排序
myList.sort();

// 翻转
myList.reverse();

// 合并另一个list（另一个list将变为空）
std::list<int> anotherList = {6, 7, 8};
myList.merge(anotherList);

// 去重
myList.unique();

      std::list 由于其底层的双向链表实现，在某些操作上比基于数组的容器（如 std::vector）更加高效，特别是在频繁插入和删除元素时。然而，这种灵活性的代价是牺牲了随机访问的性能，因此在选择容器类型时应考虑你的具体需求。

      # 2. 迭代器（Iterators）

      迭代器是一种访问容器中元素的对象，类似于指针。根据容器的类型和需要，迭代器分为多种类型，如输入迭代器、输出迭代器、前向迭代器、双向迭代器、随机访问迭代器等。

      # 3. 算法（Algorithms）

      算法是作用于容器的函数，用于执行诸如搜索、排序、计数、遍历等操作。STL的算法设计为与容器无关，可以通过迭代器与任何容器配合使用。
