# JavaScript对象和JQuery对象

用JavaScrip创造的对象叫做JavaScrip对象

用JQuery创造的对象叫做JQuery对象

JavaScript对象和JQuery对象可以相互转化，但是我们一般都使用JQuery对象，因为使用JQuery对象比较简单

# 获取JQuery

一般JQUery有两个文件，一个是未经压缩的文件，这个文件可以方便我们阅读源码，另外一个文件经过压缩，删除了空格换行等冗余符号，所以文件体积更小，方便在互联网上传播。

# 函数：val text attr

val

alert($(”#id”).val())———打印DOM对象中的值

$(”:text”).val(”HelloWorld”)———将DOM对象中的val值设置为“HelloWorld”

text

alert($(”div”).text())———将DOM对象的值全部连接打印

$(”div”).text(”hellowolrd”)———将DOM对象的val值全部设置为”hellowolrd”

attr——attribute n.属性

$(”#id”).attr(”key”)——获取dom对象中key属性的value值 $(”#id”).attr(”key”，“value”)———将dom对象中key熟悉的value值设置为value

# on（）绑定事件

语法：$(”dom对象”).on(envent,data,function)

envent表示事件，可以指一个也可以指多个

data：可选。规定传到function中的额外数据，要是json格式的

function：可选，指envent发生之后运行的函数

# $.ajax()

作用：$.ajax()可以简化复杂的异步请求对象的引用，降低代码的开发难度。

$.ajax({

URL : Servlet请求的servlet地址

type:”get” 或者 type:”post”（默认是get）

data : “{jaon、string、数组}” 表示向servlet传入的数据，可以是string、json、或者是数组

success(result,status,xhr):function 当请求成功时运行的函数

async : true 或者 async : true 表示异步对象是否开启（默认是true）

dataTypt : “json” 期望通服务器响应的数据类型。xml、html、text、json

error(xhr,status,xhr):function当请求失败时要运行的函数（默认不填）

})

# $.get()

语法：$.get(url,data,function(),dataType)

# $.post()

语法：$.post(url,data,function(),dataType)
