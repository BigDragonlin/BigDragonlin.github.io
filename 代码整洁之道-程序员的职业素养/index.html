# 测试驱动开发的好处

  • 测试驱动开发能够驱除我们对更改代码后未知后果的恐惧，使得我们能够有勇气去重构老代码，从而提高代码质量。

  • 先编写测试的习惯能够让业务代码解耦，方便我们测试，从而提高代码的可维护性。

# 练习

  • 程序员要刷题，不断锻炼自己解决新问题的能力
  • 如果你只做某一方面的某一个问题，你的思维和能力将会固化，跟不上时代的发展。你可以尝试参加一些开源项目 如果你一直写C++，你可以参加一些python开源项目的贡献。
